Frequently Asked Questions

How do I convert men's EU shoe size to US?

Use the Shoe Size Converter or the Men's Shoe Size Converter. For a single size, use the size-pair links above (e.g. EU 42 to US).

Are men's and women's shoe sizes the same in US?

No. Men's and women's US sizes use different scales. Always use the men's converter or men's size-pair pages for men's shoes.

Where is the main shoe size converter?

Use our main Shoe Size Converter or the Men's Shoe Size Converter for gender-specific conversion.
